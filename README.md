# 👥 HR Employee Attrition Analysis — SQL Project

> 🔍 A data analysis project uncovering why employees leave — using SQL to identify patterns in turnover, salary, and work-life balance across departments.

---

## 🧠 About This Project

Employee attrition is one of the most costly challenges for any organization. Losing talent means losing knowledge, productivity, and money.

This project uses **SQL** to analyze a real-world HR dataset and answer key business questions:

- 🔴 Which departments have the highest attrition rate?
- 💰 Does salary level affect the likelihood of leaving?
- 🎂 Which age groups are most at risk of leaving?
- ⚖️ Does work-life balance impact employee retention?
- 🏆 Does job satisfaction make a difference?

## 🔑 Key Business Questions

| # | Question | SQL Concept Used |
|---|----------|-----------------|
| 1 | What is the overall attrition rate? | COUNT, AVG |
| 2 | Which department loses the most employees? | GROUP BY, ORDER BY |
| 3 | Does monthly income affect attrition? | CASE, AVG, GROUP BY |
| 4 | Which age group leaves the most? | CASE, GROUP BY |
| 5 | Does work-life balance matter? | GROUP BY, AVG |
| 6 | What role does job satisfaction play? | JOIN, GROUP BY |
